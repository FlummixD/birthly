# Dynamische werte:

| Wert | Bedeutung | Benutzung |
| - | - | -|
| rem | Relative Schriftgröße | Für Padding / Margin
| vw | Bildschirmbreite | Für Größen von Containern


## Für Padding / Margin:
Abstand < 1rem für Nahgelegende Gruppierte Elemente <br>
Abstand > 1rem für Nicht Gruppierte Elemente.
